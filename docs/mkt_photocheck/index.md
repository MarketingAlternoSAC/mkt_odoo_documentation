# Módulo de Photocheck

El módulo `mkt_photocheck` está diseñado para gestionar la creación, aprobación y emisión de fotochecks (carnets de identificación) para los colaboradores. 

Este sistema permite centralizar las solicitudes, automatizar la obtención de datos mediante el DNI y facilitar el procesamiento de imágenes para asegurar un estándar de calidad en las identificaciones.

## Características Principales

*   **Gestión de Solicitudes:** Seguimiento del ciclo de vida de un fotocheck desde el borrador hasta su emisión.
*   **Integración con API DNI:** Autocompletado de nombres y apellidos al ingresar el número de DNI.
*   **Procesamiento de Imágenes:** Eliminación automática de fondo en las fotografías para cumplir con los requisitos estéticos.
*   **Control de Acceso Multinivel:** Sistema de permisos basado en marcas, ciudades y roles de usuario.
*   **Notificaciones Automáticas:** Envío de correos electrónicos informando sobre los cambios de estado en las solicitudes.
