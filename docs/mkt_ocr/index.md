# Documentación del Módulo: mkt_ocr

Este módulo integra funcionalidades de Reconocimiento Óptico de Caracteres (OCR) dentro de Odoo 17, permitiendo extraer automáticamente datos de documentos (como PDF o imágenes) para rellenar campos en el sistema. Además, incluye un sistema de registro (log) y encuestas de satisfacción para medir el rendimiento y uso del OCR.
