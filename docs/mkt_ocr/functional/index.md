# Guía Funcional: OCR

Esta sección está dirigida a los usuarios que interactúan con las funcionalidades de extracción de datos en Odoo.

## Objetivos del Módulo

El objetivo principal es reducir la entrada manual de datos de facturas y recibos físicos o en PDF, automatizando el llenado de formularios mediante Reconocimiento Óptico de Caracteres (OCR).

## Funcionalidades Clave

### Procesamiento de Documentos
Al adjuntar un documento (imagen o PDF) en un registro compatible (ej. Liquidaciones), el usuario puede procesar el archivo. El sistema extraerá los datos clave (RUC, número de documento, fecha, montos) y autocompletará los campos correspondientes.

### Evaluación de Calidad
Tras el procesamiento, el usuario puede evaluar si la extracción fue exitosa, lo que alimenta los registros para futuras mejoras.
